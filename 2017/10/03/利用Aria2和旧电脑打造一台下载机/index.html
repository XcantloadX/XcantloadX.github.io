利用Aria2和旧电脑打造一台下载机

发表于 2017-10-03 | 分类于 折腾 | 阅读次数:

利用这个下载神器—— Aria2，可以把你的旧电脑 / 手机变成一台免费的（电费除外）的 24 小时运行的下载机，比那些所谓的“离线下载”好多了

Aria2是全平台的，Windows、MAC、Linux、安卓等等。本教程只提供Win的教程，其他系统你们自己捣鼓去
安卓教程正在编写

电脑部分

下载并运行 Aria2

（为什么不写安装并配置 Aria2？因为懒人包不需要安装和配置，直接下载运行就行了）

找台 Win 7 以上 的电脑，下载这个：Aria2 懒人包
Win XP 在这里下载Windows downloads下的第一个文件，解压后把里面的aria2c.exe替换到懒人包里

解压，可以看见这些文件

双击运行.bat，Aria2 就开始运行了
同理双击停止.bat就可以关掉 Aria2，当然，直接叉掉也可以，没什么影响，因为停止.bat的原理就是结束aria2.exe

打开 Aria2 控制台 来查看效果，如果没有提示Aria2 PRC 服务器错误就表示一切正常

FTP 服务器

假如文件下载完之后要传到手机上，总不可能再插根数据线来传吧？
我们可以用FTPServer这个软件来新建一个 FTP 服务器

软件已经打包到懒人包里，使用方法非常简单：
打开 FTPServer，设置好共享目录，点启动，Win 端就配置好了，手机端在后面会讲

手机部分

安装并配置Yandex Browser

到应用市场下载Yandex Browser

用 Yandex Browser 打开 Tampermonkey Chrome网上应用店 ，别忘了挂上你的梯子（不懂的百度V P N）

安装~

安装之后，它会弹出来一个网页，不管它

访问 Greasy Fork ，直接点主页上的baidu.com，你会看到一大堆脚本

找到EX-百度云盘，点开，安装

获取链接

打开你的百度云客户端，找到你要下载的文件，分享，然后复制链接
（推荐选 1 天后失效，可以避免忘记取消分享的情况，如果是重要的文件，最好设密码）

在 Yandex Browser 里打开刚刚复制的链接，选中你要下载的文件（如果有多个文件的话），你会发现在原来的下载按钮旁会多出来一个EX-下载

确认你没有登陆之后，点一下EX-下载，再选择复制链接，复制


添加任务

用浏览器打开 Aria2 控制台 ，先不着急下载，点右上角的设置（像扳手一样的），第一项里面默认是http://localhost:6800/jsonrpc
我们把它改为http://你的内网IP:6800/jsonrpc（如何查看IP？），假如我的内网IP是192.168.0.101就改为http://192.168.0.101:6800/jsonrpc

如果你家里有很多个设备的话，而且旧电脑又是笔记本，最好在路由器里面绑定MAC地址，来固定一个内网IP，在这里查看教程

然后点击右上角的添加，把你刚刚复制的链接粘贴进去，文件名、保存位置什么的不用管，唯一要看一下的是分块数量和最大连接数
理论上这两个越大，下载速度越快
推荐分别设置为50和16，最大连接数最多为16，分块数量无限制

然后愉快地看着它高速的下载吧，别忘了回来评论和感谢我~


取回文件

下好了之后，文件是在电脑上，怎么取回来呢？看你主要下什么东西、有什么设备，有两种方案

1.使用 FTP 服务器

好处：简单、可设密码（Win文件共享也可以）、速度快（视具体情况）、外网可访问（具体请百度）
缺点：搭建需要软件支持（Win server 系统除外）
（过于简单，这里不再多说，自行百度手机如何连接 FTP 服务器）

2.Windows 文件共享

好处：更适合视频文件、更适合组建家庭媒体中心
缺点：安卓支持读写的应用不多、搭建比较过程复杂、速度不太稳定（视具体情况）

教程：
右键此电脑（计算机），打开管理，按图片打开用户

双击Guest，把账户已禁用勾掉，保存

打开网络与共享中心（控制面板找），找到高级共享设置，往下拉，找到所有网络，展开（点一下右边的倒箭头），找到最后一项密码保护的共享，关掉，保存更改

找到Aria2目录下的Download文件夹，右击，打开属性，切换到共享选项卡，点击共享按钮

打开列表，选Guest，点击添加

把权限级别改为读取/写入，保存

接下来换到手机，在手机上安装ES文件浏览器，打开
打开侧边栏，点击局域网

点击下面的扫描，然后隐藏

不一会，你的电脑就扫描出来了，点击进去
如果提示登陆，勾选匿名，直接登陆

可以看到，里面有刚刚的Download文件夹，然后把你的文件复制出来就行啦
